Task Detail & Editing

View complete task details, edit properties, manage status/priority, and delete tasks.

Updated: February 17, 2026 | Interactive flow format

Flow Steps

1
Action

Click on any task card in the chat view (e.g., "Set up design system tokens") to open task details.

Expected Result

Task detail drawer slides in from the bottom (mobile) or right side (desktop). Displays complete task information including: Task title, Creator info with creation timestamp, Full description, Status dropdown (with current status shown), Priority dropdown (with current priority), Due date (with overdue/due soon indicators if applicable), Assignee avatars, Tags as badges, Dependencies section (if any), Subtasks section with progress bar, Edit button (pencil icon), Delete button (trash icon). Drawer has scrollable content if task details are extensive.

Panel block represents task details. Edit labels to iterate field order/copy.

Key Observations

  • Quick Status Changes: Status can be changed directly from the detail view via dropdown without opening edit form
  • Priority Updates: Priority can be updated independently with a simple dropdown interaction
  • Visual Status Indicators: Each status has a unique icon and color (To Do: circle, In Progress: spinner, Review: eye, Done: checkmark, Blocked: X)
  • Priority Color Coding: Low (gray), Medium (blue), High (orange), Urgent (red)
  • Due Date Warnings: Overdue dates shown in red, due soon dates in yellow with visual indicators
  • Subtask Management: Progress bar shows completion ratio (e.g., "2/5 subtasks completed")
  • Dependency Links: Clickable dependency links navigate to related task details
  • Edit Mode: Clicking edit icon opens the full task form pre-populated with current values
  • Delete Confirmation: Delete action likely triggers confirmation dialog to prevent accidental deletion
  • Activity Tracking: Status and priority changes create system messages in the chat feed
  • Responsive Design: Drawer adjusts size and position based on screen size (85vh height on mobile)
  • Tag Display: Tags shown as small badge chips with consistent styling
