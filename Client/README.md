# ClientV2

TaskChat migration target built with Nuxt 4, Vue 3, and Nuxt UI 4.

## Tech Stack

- Nuxt 4 + Vue 3 Composition API
- Nuxt UI 4
- Tailwind CSS 4
- TypeScript
- Vitest + Nuxt test utils

## Setup

```bash
pnpm install
```

## Development

Start local dev server:

```bash
pnpm dev
```

## Tests

Run all tests:

```bash
pnpm test
```

Run targeted suites:

```bash
pnpm test:unit
pnpm test:nuxt
pnpm test:e2e
```

## Build

```bash
pnpm build
pnpm preview
```
